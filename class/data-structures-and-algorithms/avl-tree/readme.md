
# Cây cân bằng

## Quy ước

 * Trái < Gốc < Phải

 > Với mọi **node** thuộc cây, gốc của **nhánh trái** có giá trị **nhỏ hơn** giá trị của **node** và cùng **nhỏ hơn** giá trị của gốc của **nhánh phải**

 * Trường `bal` của `Node`

 > Lệch trái: `bal` = **-1**

 > Không lệch: `bal` = **0**

 > Lệch phải: `bal` = **+1**
