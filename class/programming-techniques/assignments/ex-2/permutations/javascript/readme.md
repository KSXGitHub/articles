
# Bộ tạo danh sách hoán vị dành cho JavaScript

## Yêu cầu

 * Trình duyệt Chrome hoặc Cốc Cốc phiên bản 50.0.0 hoặc mới hơn

## Cách chạy

### Chuẩn bị

 1. Mở trình duyệt, try cập vào trang [about:blank](about:blank) để mở trang trống

 2. Nhấn F12 để mở Developer Tools

 3. Chọn tag 'Console' trên Developer Tools

 4. Copy nội dung của [./code.js](./code.js) rồi paste vào Developer Tools và nhấn Enter

### Xem kết quả

Nhập `permutations(<các-phần-tử>)` rồi nhấn Enter để xem kết quả, ví dụ:

```javascript
permutations(0, 1, 2, 3)
```
